# Welcome to Cloud Note

This is your first Git-backed note.

Try editing this file. The app will autosave and create a GitHub commit in the background.

## Suggested folders

- `notes/` for normal knowledge notes
- `workflows/` for workflow design
- `agents/` for agent instructions and memory
- `.cloudnote/` for app metadata

## Why this exists

Cloud Note gives non-technical users a simple note editor while keeping GitHub as the source of truth for versioning, rollback, and agent workflows.
